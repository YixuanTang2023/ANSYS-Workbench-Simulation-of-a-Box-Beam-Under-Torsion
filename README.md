# Box Beam Torsion Simulation in ANSYS Workbench

This repository contains the ANSYS Workbench project for torsional analysis
of a fixed-free box beam. 

Files included:
- IBeam.wbpj (Workbench project)
- IBeam_files/ (project data, loads, mesh, solver files)

How to use:
1. Download the repo
2. Extract all files
3. Open IBeam.wbpj in ANSYS Workbench
